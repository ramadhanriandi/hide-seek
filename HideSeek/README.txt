Cara Penggunaan :
1. Tekan tombol 'Choose' untuk memilih file graf
2. Setelah terbaca, tekan tombol 'Generate' untuk membangun graf
3. Jika ingin memasukkan query secara langsung, ketika di textBox dan tekan tombol 'OK'
4. Jika ingin memasukkan query dari file eksternal, tekan tombol 'Choose' untuk memilih file query
5. Tekan tombol 'Solve' untuk menampilkan hasil
6. Tekan salah satu hasil di list box untuk melihat jalur yang dilalui pada graf
